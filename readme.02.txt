Este programa permite registrar empleados con su salario y luego mostrar la lista de empleados registrados. Utiliza una clase para representar a los empleados, una lista para almacenar los empleados y un bucle para el menú principal del programa. También utiliza la entrada/salida de la consola para interactuar con el usuario. Por favor, ten en cuenta que este es un programa de consola y debe ser ejecutado en un entorno que soporte la entrada/salida de la consola.
